# Module 2 Implementation Plan
## Goals
- Control Module
- Lane Keep Assistant (LKA)
- Basic Cruise Control (CC)
- Bonus: Adaptative Cruise Control (ACC)
- Object Detection and Avoidance (ODA)
- Emergency Brake Assist (EBA)
- Bonus: Truck Platooning
- Traffic Sign Recognition
- Requirements with TSF: Pull Request to coursebook
- ADR for the Models Selection Decision
- Qt to integrate the new ADAS features.
- Unit and Integration Tests
- Architecture Decision Record (ADR)

### Phase 1: Foundation and Basic Lane Navigation
- Control Model (PID + MPC)
- CARLA Simulator Setup
- LKA — Lane Keeping Assist
- EBA — Emergency Brake Assist
- TSF Requirements & ADR
- Unit Test Infrastructure

### Phase 2: Advanced Decision Making and Traffic Sign Recognition
- CC — Cruise Control
- Traffic Sign Recognition (TSR) Model
- TSR Integration with Speed Control
- Qt Cluster UI — ADAS Overlay
- Bonus Implementation : Truck Platooning, ACC, ...
