## Motors

## Back Motors

```math
V = 6~15V
Running I = 120mA at (12V)
Stall I = 1.2A at (12V)
Running P = 12 * 0,12 = 1,44W
Stall P = 12 * 1,2 = 14,4W
```

## Servo Motor

```math
V = 4.8V~7.2V
Running I = 500mA
Stall I = 2.5A at (6V)
Running P = 5V * 0,500 = 2,5W
Stall P = 5V * 1,2 = 6W
```

## TouchScreen

```math
V = 5V

```
