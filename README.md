# Team2 SEA:ME Project | Autonomous Driving
This repo contains all the software and documentation we use to achieve autonomous driving in a PiRacer

# Overview
This project shows an autonomous driving system built with a PiRacer Kit, Raspberry Pi 5
machine learning

# Hardware Setup
- PiRacer Kit (Waveshare)
- 7.9 inch Waveshare Display
- Raspberry Pi 5
- Raspberry Pi AI HAT+ 26 Tops
- Raspberry Pi SSD Kit
- STM32 MCU (B-U585I-IOT02A)
- MCP2515/TJA1050 (CAN Controller and Transceiver)
